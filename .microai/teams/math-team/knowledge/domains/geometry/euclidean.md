# Hình học Euclid

## Tam giác

### Định lý cơ bản
```latex
\text{Tổng 3 góc: } \alpha + \beta + \gamma = 180°

\text{Bất đẳng thức tam giác: } |a - b| < c < a + b
```

### Công thức diện tích
```latex
S = \frac{1}{2}ah \quad \text{(cạnh × đường cao)}

S = \frac{1}{2}ab\sin C \quad \text{(2 cạnh và góc xen)}

S = \sqrt{p(p-a)(p-b)(p-c)} \quad \text{(Heron)}

S = \frac{abc}{4R} \quad \text{(3 cạnh và bán kính ngoại tiếp)}

S = pr \quad \text{(nửa chu vi × bán kính nội tiếp)}
```

### Định lý sin
```latex
\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R
```

### Định lý cos
```latex
c^2 = a^2 + b^2 - 2ab\cos C
```

### Đường trung tuyến
```latex
m_a^2 = \frac{2b^2 + 2c^2 - a^2}{4}
```

### Đường phân giác
```latex
l_a = \frac{2bc\cos\frac{A}{2}}{b + c}
```

## Tam giác đặc biệt

### Tam giác vuông
```latex
\text{Pythagore: } a^2 + b^2 = c^2

\text{Đường cao: } h^2 = a' \cdot b'

\text{Hệ thức lượng: } a^2 = a' \cdot c
```

### Tam giác cân
- Hai cạnh bên bằng nhau
- Hai góc đáy bằng nhau
- Đường cao = đường trung tuyến = đường phân giác (từ đỉnh)

### Tam giác đều
```latex
S = \frac{a^2\sqrt{3}}{4}

R = \frac{a\sqrt{3}}{3}

r = \frac{a\sqrt{3}}{6}
```

## Đường tròn

### Công thức
```latex
\text{Chu vi: } C = 2\pi R

\text{Diện tích: } S = \pi R^2

\text{Độ dài cung: } l = \frac{\alpha \cdot \pi R}{180°}

\text{Diện tích quạt: } S = \frac{\alpha \cdot \pi R^2}{360°}
```

### Góc và cung
```latex
\text{Góc ở tâm = cung bị chắn}

\text{Góc nội tiếp} = \frac{1}{2} \text{cung bị chắn}

\text{Tứ giác nội tiếp: } \alpha + \gamma = \beta + \delta = 180°
```

### Tiếp tuyến
- Tiếp tuyến vuông góc với bán kính tại tiếp điểm
- Hai tiếp tuyến từ một điểm ngoài có độ dài bằng nhau

### Dây cung
```latex
\text{Đường kính qua trung điểm dây vuông góc với dây}

d^2 = R^2 - \frac{l^2}{4} \quad \text{(d: khoảng cách tâm-dây, l: độ dài dây)}
```

## Tứ giác

### Hình bình hành
- Hai đường chéo cắt nhau tại trung điểm mỗi đường
- Hai cặp cạnh đối song song và bằng nhau

### Hình chữ nhật
- Hình bình hành có một góc vuông
- Hai đường chéo bằng nhau

### Hình thoi
- Hình bình hành có 2 cạnh kề bằng nhau
- Hai đường chéo vuông góc

### Hình vuông
- Vừa là HCN vừa là hình thoi
- Hai đường chéo bằng nhau và vuông góc

### Hình thang
```latex
S = \frac{1}{2}(a + b)h \quad \text{(a, b: 2 đáy, h: chiều cao)}

\text{Đường trung bình} = \frac{a + b}{2}
```

## Các định lý quan trọng

### Thales
Đường thẳng song song với một cạnh tam giác cắt hai cạnh còn lại thì tạo tỉ lệ bằng nhau.

### Đồng dạng tam giác
- c-c-c: Ba cạnh tỉ lệ
- c-g-c: Hai cạnh tỉ lệ và góc xen bằng nhau
- g-g: Hai góc bằng nhau

### Bằng nhau tam giác

- c-g-c
