> Knowledge file cho Script Writer Agent

## Format 1: Vocabulary Shorts (30 giây)

### Template Structure

```markdown
# Script: {word} - 30s Vocabulary Shorts

## Metadata
- Word: {word}
- IPA: {ipa}
- POS: {part_of_speech}
- Duration: 30 seconds
- Format: 9:16 Shorts

---

## Script

### [0:00-0:03] HOOK
**Audio (VI):** "Từ này 90% người Việt phát âm sai!"
**Visual:**
- Text: "90% SAI" (large, attention-grabbing)
- Animation: Zoom in effect
- Background: Brand color gradient

### [0:03-0:08] WORD INTRO
**Audio (EN):** "{word}"
**Audio (EN):** "/{ipa}/"
**Visual:**
- Text main: "{word}" (large, centered)
- Text sub: "/{ipa}/ - {pos}"
- Animation: Fade in with highlight

### [0:08-0:15] DEFINITION
**Audio (VI):** "Nghĩa là {meaning_vi}"
**Visual:**
- Text main: "{meaning_vi}"
- Text sub: "{meaning_en}"
- Icon: Definition icon

### [0:15-0:22] EXAMPLE
**Audio (EN):** "{example_sentence_en}"
**Audio (VI):** "{example_sentence_vi}"
**Visual:**
- Text: Example sentence with {word} highlighted
- Animation: Word highlight pulse

### [0:22-0:27] TIP
**Audio (VI):** "Nhớ:"
**Audio (EN):** "{collocation}"
**Visual:**
- Text: "Collocation"
- Text sub: "{collocation}"
- Icon: Lightbulb

### [0:27-0:30] CTA
**Audio (VI):** "Follow để học mỗi ngày!"
**Visual:**
- Text: "FOLLOW"
- Text sub: "Học TOEIC mỗi ngày"
- Animation: Pulse effect
```

## Format 2: Grammar Shorts (45 giây)

### Template Structure

```markdown
# Script: {grammar_rule} - 45s Grammar Shorts

## Metadata
- Rule: {grammar_rule}
- Part: {toeic_part}
- Duration: 45 seconds
- Format: 9:16 Shorts

---

## Script

### [0:00-0:05] HOOK
**Audio (VI):** "Bạn có mắc lỗi này không?"
**Visual:**
- Wrong example (crossed out)
- Text: "SAI LẦM PHỔ BIẾN"

### [0:05-0:12] WRONG EXAMPLE
**Audio (EN):** "{wrong_sentence}" ❌
**Audio (VI):** "Câu này sai ở đâu?"
**Visual:**
- Text: Wrong sentence
- Animation: Red X mark

### [0:12-0:22] EXPLANATION
**Audio (VI):** "{explanation_vi}"
**Visual:**
- Text: Rule explanation
- Diagram: Grammar structure

### [0:22-0:32] CORRECT EXAMPLE
**Audio (EN):** "{correct_sentence}" ✓
**Audio (VI):** "Đây mới là cách đúng"
**Visual:**
- Text: Correct sentence
- Animation: Green checkmark

### [0:32-0:40] RULE SUMMARY
**Audio (VI):** "Quy tắc: {rule_summary}"
**Visual:**
- Rule box with key points
- Bullet points

### [0:40-0:45] CTA
**Audio (VI):** "Save lại để nhớ! Follow để học thêm!"
**Visual:**
- Save + Follow icons
- Brand outro
```

## Format 3: Listening Standard (3-5 phút)

### Template Structure

```markdown
# Script: {topic} - Listening Practice

## Metadata
- Part: {toeic_part}
- Topic: {topic}
- Duration: 3-5 minutes
- Format: 16:9 Standard

---

## Script

### [0:00-0:15] INTRO
**Audio (VI):** "Xin chào! Hôm nay chúng ta sẽ luyện {part} với chủ đề {topic}"
**Visual:**
- Channel intro
- Topic title
- Part indicator

### [0:15-0:45] STRATEGY
**Audio (VI):** "Trước khi nghe, hãy nhớ các tips sau:"
- Tip 1: {listening_tip_1}
- Tip 2: {listening_tip_2}
- Tip 3: {listening_tip_3}
**Visual:**
- Tips list with icons
- Key phrases to listen for

### [0:45-1:30] PRACTICE - FIRST LISTEN
**Audio (VI):** "Bây giờ, hãy nghe đoạn hội thoại sau"
**Audio (EN):** {conversation_audio}
**Visual:**
- "LISTENING..." indicator
- Question preview (optional)

### [1:30-2:00] QUESTIONS
**Audio (VI):** "Hãy trả lời các câu hỏi sau:"
- Q1: {question_1} (A) (B) (C) (D)
- Q2: {question_2} (A) (B) (C) (D)
- Q3: {question_3} (A) (B) (C) (D)
**Visual:**
- Questions with answer choices
- Timer countdown

### [2:00-3:30] ANSWER ANALYSIS
**Audio (VI):**
- "Câu 1: Đáp án đúng là {answer_1} vì {explanation_1}"
- "Câu 2: Đáp án đúng là {answer_2} vì {explanation_2}"
- "Câu 3: Đáp án đúng là {answer_3} vì {explanation_3}"
**Visual:**
- Answer reveal with explanation
- Key phrases highlighted in transcript

### [3:30-4:00] TRANSCRIPT REVIEW
**Audio (EN):** {conversation_audio} (slower, with pauses)
**Visual:**
- Full transcript with highlights
- Key vocabulary annotated

### [4:00-4:30] KEY TAKEAWAYS
**Audio (VI):** "Những điểm cần nhớ:"
- Takeaway 1
- Takeaway 2
- Vocabulary to remember
**Visual:**
- Summary card
- Vocabulary list

### [4:30-5:00] OUTRO
**Audio (VI):** "Cảm ơn bạn đã xem! Đừng quên like, subscribe và bật chuông thông báo!"
**Visual:**
- Subscribe animation
- Next video suggestion
- End screen
```

## Script Writing Guidelines

### Language Tagging

```yaml
language_tags:
  english:
    - words (vocabulary)
    - ipa_pronunciation
    - example_sentences
    - collocations
    - linguistic_terms

  vietnamese:
    - explanations
    - translations
    - instructions
    - cta_calls
    - tips
```

### Timing Rules

```yaml
timing:
  shorts:
    hook: 3-5s
    main_content: 20-50s
    cta: 3-5s
    total_max: 60s

  standard:
    intro: 15-30s
    main_content: 2-8min
    outro: 15-30s
```

### Visual Cues Format

```yaml
visual_cue_format:
  text_main: "Primary text shown on screen"
  text_sub: "Secondary/subtitle text"
  animation: fade_in|zoom|pulse|highlight
  icon: lightbulb|checkmark|x_mark|arrow
  transition: cut|fade|slide
```

---

## Output Format

Script files should be saved as:
- `script.md` - Main script file
- `visual-cues.json` - Structured visual instructions
- `audio-segments.yaml` - Audio segment breakdown
