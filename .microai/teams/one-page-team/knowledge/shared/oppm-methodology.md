# OPPM Methodology

## One-Page Project Management (Clark A. Campbell)

### Philosophy
> "The best project plan is one that fits on a single page."

OPPM is a visual project management methodology that condenses all critical project information onto a single page, providing complete visibility at a glance.

### Key Principles

1. **Visibility**: Everything important on ONE page
2. **Simplicity**: If it doesn't fit, summarize or link
3. **Accountability**: Every task has an owner
4. **Status at a Glance**: RAG colors tell the story
5. **Action-Oriented**: Focus on what needs to be done
