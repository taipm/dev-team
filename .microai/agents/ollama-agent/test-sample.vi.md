# Bắt đầu

Chào mừng đến với tài liệu hướng dẫn dự án.

## Yêu cầu ban đầu

Trước khi bắt đầu, hãy đảm bảo bạn đã có:

- Node.js 18 hoặc cao hơn
- npm hoặc yarn được cài đặt
- Một trình soạn thảo mã như VS Code

## Cài đặt

Để cài đặt gói, hãy chạy:

```bash
npm install my-package
```

## Sử dụng

Nhập module trong mã nguồn:

```javascript
import { MyClass } from 'my-package';

const instance = new MyClass();
instance.doSomething();
```

> **Lưu ý:** Tính năng này đang trong giai đoạn thử nghiệm.

## Tính năng

| Tính năng | Mô tả |
|-----------|-------|
| Nhanh | Tốc độ chạy cực nhanh |
| Dễ | Giao diện API đơn giản |
| An toàn | Các tính năng bảo mật tích hợp |

Để biết thêm thông tin, xem [Tài liệu API](./api.md).
