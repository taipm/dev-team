## 10. Usage Patterns

### Pattern 1: First mention expanded

```
Original: "Clone the repository to your local machine."
First: "Clone (sao chép) repository về máy local."
After: "Clone repository..."
```

### Pattern 2: Context determines translation

```
CLI context: "Run git branch to list branches"
→ "Chạy git branch để liệt kê các branch"

Explanation context: "Create a new branch for your feature"
→ "Tạo một nhánh mới cho tính năng của bạn"
```

### Pattern 3: Keep technical, translate descriptive

```
Original: "The commit message should be descriptive"
→ "Commit message nên mô tả rõ ràng"
         ↑ giữ          ↑ dịch
```
