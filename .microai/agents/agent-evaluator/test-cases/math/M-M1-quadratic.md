# Quadratic Equation

## Prompt

<prompt>
Giải phương trình bậc 2: x² - 5x + 6 = 0

Tìm tất cả các nghiệm.
</prompt>

## Expected Answer

```
Cách 1: Phân tích
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
x = 2 hoặc x = 3

Cách 2: Công thức nghiệm
a = 1, b = -5, c = 6
Δ = b² - 4ac = 25 - 24 = 1 > 0
x = (5 ± 1) / 2
x₁ = 3, x₂ = 2
```
