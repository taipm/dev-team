# Trigonometry Basics

## Prompt

<prompt>
Trong tam giác vuông ABC vuông tại A, biết BC = 10cm, góc B = 30°.

a) Tính cạnh AC (đối diện góc B)
b) Tính cạnh AB

Sử dụng các giá trị: sin30° = 0.5, cos30° = √3/2 ≈ 0.866
</prompt>

## Expected Answer

```
a) sin B = AC/BC
   sin 30° = AC/10
   AC = 10 × 0.5 = 5 cm

b) cos B = AB/BC
   cos 30° = AB/10
   AB = 10 × 0.866 ≈ 8.66 cm

   Hoặc: AB = 10 × (√3/2) = 5√3 cm
```
