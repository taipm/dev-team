# Linear Equation

## Prompt

<prompt>
Giải phương trình: 3x + 7 = 2x + 12

Tìm giá trị của x.
</prompt>

## Expected Answer

```
3x + 7 = 2x + 12
3x - 2x = 12 - 7
x = 5
```
