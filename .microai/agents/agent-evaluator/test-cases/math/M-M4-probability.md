# Basic Probability

## Prompt

<prompt>
Gieo một con xúc xắc cân đối.
Tính xác suất để được mặt 6 chấm?

Giải thích cách tính.
</prompt>

## Expected Answer

```
Không gian mẫu: Ω = {1, 2, 3, 4, 5, 6}
Số phần tử: n(Ω) = 6

Biến cố A: "Được mặt 6 chấm"
A = {6}
n(A) = 1

Xác suất: P(A) = n(A)/n(Ω) = 1/6
```
