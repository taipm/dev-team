# Arithmetic Sequence

## Prompt

<prompt>
Cho dãy số cộng: 2, 5, 8, 11, ...

a) Tìm công sai d
b) Tìm số hạng thứ 17 của dãy
</prompt>

## Expected Answer

```
a) Công sai d = 5 - 2 = 3

b) Công thức: aₙ = a₁ + (n-1)d
   a₁₇ = 2 + (17-1) × 3
   a₁₇ = 2 + 48
   a₁₇ = 50
```
