# Percentage Calculation

## Prompt

<prompt>
Một mặt hàng giá gốc 150.000đ được giảm 10%.
Tính giá sau khi giảm?
</prompt>

## Expected Answer

```
Số tiền giảm = 150.000 × 10% = 15.000đ
Giá sau giảm = 150.000 - 15.000 = 135.000đ

Hoặc: 150.000 × (100% - 10%) = 150.000 × 90% = 135.000đ
```
