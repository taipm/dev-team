# Functions and Graphs

## Prompt

<prompt>
Cho hàm số y = x² + 2x + 5

a) Xác định tọa độ đỉnh của parabol
b) Hàm số đạt giá trị nhỏ nhất tại đâu? Giá trị đó là bao nhiêu?
</prompt>

## Expected Answer

```
y = x² + 2x + 5
y = (x² + 2x + 1) + 4
y = (x + 1)² + 4

a) Đỉnh parabol: I(-1, 4)
   (vì (x+1)² ≥ 0 với mọi x, đạt giá trị 0 khi x = -1)

b) Hàm số đạt giá trị nhỏ nhất khi x = -1
   Giá trị nhỏ nhất: ymin = 4
```

## Rubric

| Score | Criteria |
|-------|----------|
| 5 pts | Đỉnh (-1, 4), ymin=4, biến đổi đầy đủ |
| 4 pts | Đáp án đúng, thiếu biến đổi |
| 3 pts | Đúng x=-1, sai y hoặc ngược lại |
| 2 pts | Hiểu cách tìm đỉnh nhưng tính sai |
| 1 pt  | Biết là parabol nhưng không tìm được |
| 0 pts | Không hiểu hàm bậc 2 |
