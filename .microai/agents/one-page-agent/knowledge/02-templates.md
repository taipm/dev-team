## Customization Tips

### Adding Sections

```
Available sections you can add:
- DEPENDENCIES: Critical dependencies between tasks
- STAKEHOLDERS: Key stakeholders and communication plan
- ASSUMPTIONS: Key assumptions the plan is based on
- DECISIONS: Important decisions made
```

### Removing Sections

```
Sections you can remove if not needed:
- BUDGET (if not tracking)
- HABITS (Personal template only)
- CEREMONIES (Agile template only)
```
