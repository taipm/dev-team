## Content Best Practices

### Optimal Length
| Type | Characters | Words |
|------|------------|-------|
| Short (engagement) | 40-80 | 10-20 |
| Standard | 100-250 | 25-60 |
| Long (educational) | 300-500 | 75-125 |

### Emoji Usage
- 1-3 emojis for headers/bullets
- Avoid overuse (max 5-7 per post)
- Use relevant emojis only

### Hashtag Guidelines
- 1-3 hashtags optimal
- Place at end of post
- Mix popular + niche tags
- Avoid banned hashtags

### Image Recommendations
- Dimensions: 1200 x 630 px (landscape)
- Square: 1080 x 1080 px
- File size: < 4MB
- Format: PNG or JPG
- Include text overlay for key message

### Posting Schedule (Vietnam)
| Time | Engagement |
|------|------------|
| 7:00-9:00 | Good |
| 11:00-13:00 | Best |
| 19:00-21:00 | Best |
| 14:00-17:00 | Average |
