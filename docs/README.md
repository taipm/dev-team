# Tài liệu @microai.club/dev-team

Bộ cấu hình Claude Code cho team phát triển phần mềm.

---

## Giới thiệu

**@microai.club/dev-team** cung cấp:
- Custom agents cho các tác vụ chuyên biệt
- Skills framework để mở rộng khả năng
- Custom slash commands
- Hooks system cho automation
- Team configuration chia sẻ qua Git
