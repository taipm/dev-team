# Danh Sách Commands

Tất cả slash commands có sẵn.

## Agents

| Command | Agent | Mô tả |
|---------|-------|-------|
| `/microai:father` | father-agent | Tạo agents mới |
| `/microai:npm` | npm-agent | npm package management |
| `/microai:deep-question` | deep-question-agent | Đặt câu hỏi sâu |

## Go Development

| Command | Agent | Mô tả |
|---------|-------|-------|
| `/microai:go:go-dev` | go-dev-portable | Go implementation |
| `/microai:go:go-refactor` | go-refactor-portable | Go refactoring |
| `/microai:go:go-review-linus` | go-review-linus | Code review style Linus |

## Teams

| Command | Team | Mô tả |
|---------|------|-------|
| `/microai:deep-thinking` | Deep Thinking Team | 7 Titans analysis |
| `/microai:dev-user-session` | dev-user | User Story creation |
| `/microai:dev-architect-session` | dev-architect | System design |
| `/microai:dev-qa-session` | dev-qa | Testing & review |
| `/microai:dev-security-session` | dev-security | Security review |
| `/microai:pm-dev-session` | pm-dev | Requirements & estimation |
| `/microai:dev-algo-session` | dev-algo | Algorithm design |

## Specialized

| Command | Team/Agent | Mô tả |
|---------|------------|-------|
| `/microai:research` | deep-research | Deep research |

## Quick Reference

### Cho câu hỏi nhanh
```
/microai:deep-question
```

### Cho vấn đề phức tạp
```
/microai:deep-thinking
```

### Cho Go development
```
/microai:go:go-dev
```

### Cho tạo User Story
```
/microai:dev-user-session
```

### Cho system design
```
/microai:dev-architect-session
```
