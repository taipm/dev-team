# Hướng Dẫn Sử Dụng @microai.club/dev-team

Chào mừng bạn đến với **Hướng Dẫn Sử Dụng** của dev-team framework!

## Giới Thiệu

**@microai.club/dev-team** là một framework cấu hình Claude Code cho các team phát triển phần mềm. Framework này cung cấp:

- **Agents** - Các trợ lý AI chuyên biệt cho từng nhiệm vụ
- **Teams** - Nhóm agents phối hợp giải quyết vấn đề phức tạp
- **Commands** - Các lệnh slash để kích hoạt agents nhanh chóng
- **Skills** - Các kỹ năng tái sử dụng cho nhiều tình huống

## Cuốn Sách Này Dành Cho Ai?

Cuốn sách này dành cho **người dùng** muốn:

- Cài đặt và cấu hình dev-team trong dự án của mình
- Sử dụng các agents và teams có sẵn
- Áp dụng workflows hiệu quả vào công việc hàng ngày

> **Lưu ý**: Nếu bạn muốn **tạo mới** agents hoặc teams, hãy tham khảo cuốn [Developer Guide](../../developer-guide/vi/src/README.md).

## Yêu Cầu

Trước khi bắt đầu, bạn cần:

- **Claude Code** (Anthropic's CLI for Claude)
- **Node.js** >= 18.0.0
- **npm** hoặc **pnpm**
