Hướng Dẫn Sử Dụng @microai.club/dev-team

Glossary

Từ điển thuật ngữ.

A

Agent

AI assistant được cấu hình chuyên biệt với persona, knowledge, tools, và memory.

ADR (Architecture Decision Record)

Tài liệu ghi lại quyết định kiến trúc quan trọng, context, và consequences.

Acceptance Criteria

Điều kiện cần đáp ứng để User Story được coi là hoàn thành.

C

Claude Code

CLI tool của Anthropic để làm việc với Claude trong terminal.

Command (Slash Command)

Shortcut để kích hoạt agents hoặc teams, format /namespace:name.

D

Deep Thinking Team

Team của 7 Titans phân tích vấn đề phức tạp qua 5 phases.

dev-team

Framework cung cấp AI agents và teams cho software development.

F

Five Phases Protocol

Quy trình của Deep Thinking Team: Understand → Analyze → Explore → Synthesize → Validate.

First Principles

Phương pháp tư duy break down vấn đề về các nguyên lý cơ bản.

H

Handoff

Chuyển giao context và kết quả giữa các agents trong team.

K

Knowledge System

Hệ thống quản lý kiến thức của agents với knowledge-index.yaml và topic files.

M

Memory System

Hệ thống lưu trữ context, decisions, learnings qua các sessions.

MicroAI Framework

Framework 3-layer cho AI agents và teams.

O

Observer Commands

Commands điều khiển team sessions: *status, *focus, *next, v.v.

P

Phase

Giai đoạn trong workflow của team.

Persona

Vai trò và phong cách giao tiếp của agent.

S

Session Team

Team dạng dialogue turn-based giữa hai roles.

Skill

Workflow có thể tái sử dụng với references và examples.

SUMMARY.md

File định nghĩa mục lục của mdBook.

T

Team

Nhóm agents làm việc cùng nhau theo workflow.

Team Memory

Bộ nhớ chung của team với context, decisions, handoffs.

Titan

Một trong 7 experts trong Deep Thinking Team.

Tools

Các công cụ agent có thể sử dụng: Read, Write, Edit, Bash, v.v.

U

User Story

Mô tả requirement từ góc độ user: “As a… I want… So that…”

W

Workflow

Quy trình làm việc của team với các phases.

workflow.md

File định nghĩa workflow của team.

5

5W2H

Framework phân tích: What, Why, Where, When, Who, How, How much.

5 Why Analysis

Phương pháp hỏi “Tại sao?” 5 lần để tìm root cause.
