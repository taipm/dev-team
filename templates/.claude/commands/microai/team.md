# MicroAI Team Activation

Activate a specialized development team from MicroAI framework.

## Available Teams

1. **go-team** - Go development specialists (11 agents)
2. **project-team** - Backend project team (15 agents)
3. **dev-user** - User story collaboration
4. **mining-team** - Deep analysis specialists

## Usage

Specify team: $ARGUMENTS

## Team Selection

Based on your request, I will:
1. Load team configuration from `.microai/agents/microai/teams/`
2. Initialize team memory and context
3. Activate appropriate team lead/orchestrator
4. Begin collaborative workflow
