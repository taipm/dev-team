# MicroAI Kanban Management

Manage your project's Kanban board through MicroAI.

## Commands

- `show` - Display current board state
- `add <task>` - Add new task to backlog
- `move <task> <status>` - Move task to new status
- `assign <task> <agent>` - Assign task to agent

## Statuses

- `backlog` - Not started
- `todo` - Ready to work
- `in-progress` - Currently working
- `review` - Under review
- `done` - Completed

## Usage

$ARGUMENTS

## Configuration

Board configuration: `.microai/kanban/board.yaml`
