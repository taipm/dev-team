# Go Development Mode

Activate the MicroAI Go development team for your project.

## Available Sub-commands

- `/microai go dev` - Go development agent
- `/microai go refactor` - Go refactoring specialist
- `/microai go review` - Code review (Linus style)
- `/microai go team` - Full Go team orchestration

## Usage

$ARGUMENTS

## Context

Load the Go team configuration from `.microai/agents/` and begin development workflow.
