# MicroAI Commands

Slash commands for MicroAI framework integration.

## Available Commands

| Command | Description |
|---------|-------------|
| `/microai go` | Go development workflows |
| `/microai team` | Activate development teams |
| `/microai kanban` | Kanban board management |

## Usage

```
/microai go dev        # Start Go development
/microai go refactor   # Refactor Go code
/microai team go-team  # Activate Go team
/microai kanban show   # Show Kanban board
```

## Configuration

MicroAI agents and teams are configured in `.microai/` directory.

See `.microai/agents/` for available agents.
