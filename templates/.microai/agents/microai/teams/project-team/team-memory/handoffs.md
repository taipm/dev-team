## Handoff Protocol

```
BEFORE HANDOFF:
□ Document current state
□ List remaining work
□ Note any blockers
□ Commit/save all changes

DURING HANDOFF:
□ Add entry to this file
□ @mention receiving agent
□ Provide context summary

AFTER HANDOFF:
□ Receiving agent confirms
□ Mark "Handoff Verified"
```
