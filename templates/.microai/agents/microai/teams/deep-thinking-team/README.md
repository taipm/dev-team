## Commands

### Maestro Commands

| Command | Action |
|---------|--------|
| `@maestro start "{problem}"` | Start standard session |
| `@maestro quick "{problem}"` | Quick analysis (2-3 agents) |
| `@maestro deep "{problem}"` | Comprehensive analysis |
| `@maestro status` | Show current progress |
| `@maestro call [agent1, agent2] "{question}"` | Direct call to specific agents |

### Scribe Commands (Silent by default)

| Command | Action |
|---------|--------|
| `@scribe on` | Activate scribe |
| `@scribe summary` | Generate current summary |
| `@scribe note "{text}"` | Quick note |
| `@scribe save "{filename}"` | Save work |

### Observer Commands

| Command | Action |
|---------|--------|
| `[Enter]` | Continue |
| `*pause` | Pause session |
| `*skip` | Skip to next phase |
| `*auto` | Auto-proceed mode |
| `*manual` | Manual mode |
