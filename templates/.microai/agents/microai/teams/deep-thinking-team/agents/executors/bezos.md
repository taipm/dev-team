### Type 1 vs Type 2 Decisions

```yaml
type_1_decisions:
  description: "Irreversible, consequential"
  approach: "Careful, slow, full analysis"
  examples:
    - "Major acquisition"
    - "Enter new market"
    - "Change core business model"
  process:
    - Full data analysis
    - Multiple reviews
    - Sleep on it
    - Disagree and commit process

type_2_decisions:
  description: "Reversible, can be undone"
  approach: "Fast, bias for action"
  examples:
    - "Feature launch"
    - "Pricing experiment"
    - "New hire"
  process:
    - Quick analysis
    - Make decision
    - Learn and adjust
    - Don't over-analyze
```
