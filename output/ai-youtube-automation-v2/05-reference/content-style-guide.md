# Content Style Guide

## Muc dich
Dinh nghia chuan content cho channel, dam bao nhat quan va chat luong.

---

## Brand Voice

### Tone
| Do | Don't |
|-----|--------|
| Friendly, approachable | Cold, academic |
| Confident, knowledgeable | Arrogant, preachy |
| Curious, encouraging | Boring, condescending |
| Practical, actionable | Theoretical only |

### Language Style
- **Simple**: Dung tu ngu de hieu
- **Conversational**: Nhu dang noi chuyen
- **Active voice**: "Ban co the..." thay vi "Co the duoc..."
- **Second person**: "Ban", "Cua ban"

---

## Script Structure

### Standard Template (5-8 min video)

```
[HOOK - 0:00-0:30] 30 giay
- Cau hoi gay to mo hoac fact bat ngo
- Problem statement
- Promise gia tri

[INTRO - 0:30-1:00] 30 giay
- Gioi thieu topic
- Outline noi dung
- "Xem den cuoi de..."

[MAIN CONTENT - 1:00-7:00] 6 phut
- Point 1 (1-2 phut)
- Point 2 (1-2 phut)
- Point 3 (1-2 phut)
- Examples, stories, data

[CONCLUSION - 7:00-8:00] 1 phut
- Recap key points
- Call to action
- Teaser video tiep theo

[OUTRO - 8:00-8:30] 30 giay
- Subscribe reminder
- End screen pointing to next video
```

## Visual Style

### Color Palette
| Element | Color | Hex |
|---------|-------|-----|
| Primary | Orange | #FF6B35 |
| Secondary | Dark Blue | #1A1A2E |
| Accent | White | #FFFFFF |
| Text | Black/White | #000000/#FFFFFF |

### Typography
| Use | Font | Weight |
|-----|------|--------|
| Headlines | Montserrat | Bold |
| Body | Open Sans | Regular |
| Captions | Roboto | Medium |

### Thumbnail Style
1. **Text**: Max 5 tu, BOLD
2. **Colors**: High contrast
3. **Elements**: 1 focal point
4. **Faces/Emojis**: Khi phu hop
5. **Background**: Clean, simple

## Video Production Standards

### Audio
| Setting | Value |
|---------|-------|
| Format | MP3/AAC |
| Bitrate | 192kbps+ |
| Noise | Minimal |
| Volume | -12 to -6 dB |
| Background music | 20% volume |

### Video
| Setting | Value |
|---------|-------|
| Resolution | 1920x1080 (1080p) |
| FPS | 30 |
| Format | MP4 (H.264) |
| Aspect Ratio | 16:9 |

### Text Overlays
- Font size: Readable on mobile
- Duration: 3-5 seconds
- Position: Lower third or center
- Animation: Subtle fade/slide

## Content Categories

### 1. List Videos (30%)
- "5 Cach...", "10 Bi Quyet..."
- Easy to produce
- High watch time
- Good for beginners

### 2. How-To Videos (30%)
- "Cach de...", "Huong dan..."
- Practical value
- Searchable
- Builds authority

### 3. Story/Case Study (20%)
- "Tai sao...", "Cau chuyen..."
- Emotional connection
- Higher engagement
- Shareable

### 4. Facts/Explainer (20%)
- "Su that ve...", "Giai thich..."
- Educational value
- Evergreen content
- Good for SEO

## Hook Formulas

### Question Hook
```
"Ban co biet tai sao [surprising fact]?"
"Da bao gio ban tu hoi [common question]?"
```

### Statistic Hook
```
"95% nguoi [negative outcome]. Day la cach de..."
"Nghien cuu cho thay [surprising statistic]..."
```

### Story Hook
```
"Toi tung [personal experience]..."
"Co mot nguoi dan ong [story setup]..."
```

### Curiosity Hook
```
"Dieu toi sap noi se khien ban ngac nhien..."
"Day la bi mat ma [authority] khong muon ban biet..."
```

## Call-to-Action Templates

### Subscribe CTA
```
"Neu ban thay video nay huu ich, nhan SUBSCRIBE va
bieu tuong chuong de khong bo lo video moi moi tuan!"
```

### Engagement CTA
```
"Comment ben duoi video nao ban muon xem tiep theo!"
"Like video neu ban hoc duoc dieu gi moi hom nay!"
```

### Watch More CTA
```
"Xem video tiep theo de hieu sau hon ve [topic]"
"Playlist [name] co them nhieu video hay!"
```

## Quality Checklist (Per Video)

### Script
- [ ] Hook trong 10 giay dau
- [ ] Noi dung match voi title
- [ ] 3-5 key points ro rang
- [ ] CTA cuoi video
- [ ] Khong co loi chinh ta

### Audio
- [ ] Voice ro rang, khong nhieu
- [ ] Background music phu hop
- [ ] Volume can bang

### Video
- [ ] 1080p resolution
- [ ] Text doc duoc tren mobile
- [ ] Transitions smooth
- [ ] Khong co loi ky thuat

### Thumbnail
- [ ] Eye-catching
- [ ] Text lon, ro
- [ ] High contrast
- [ ] Match voi content

### SEO
- [ ] Title co keyword chinh
- [ ] Description day du
- [ ] 15+ relevant tags
- [ ] Hashtags trong description
